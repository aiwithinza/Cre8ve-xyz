# Cre8ve.xyz - Brand Book & Design System

## 1. Typography
The primary typographic scale relies on contrasting a geometric sans-serif for headings with a highly legible sans-serif for body copy, and a monospace font for technical accents.

*   **Headings:** Space Grotesk
*   **Body Text:** Inter
*   **Technical / Accent Font:** JetBrains Mono

## 2. Color System
The color palette is built for a dark AI-centric interface, utilizing high-contrast accents against a deep dark background.

**Primary Palette:**
*   **Pumpkin Orange:** `#FF7A1A`
*   **Steel Blue:** `#4682B4`
*   **Deep Black:** `#050505`
*   **Pure White:** `#FFFFFF`

**Gradient System:**
*   Pumpkin Orange → Steel Blue
*   Steel Blue → Deep Black

## 3. Logo System
*   **Primary Wordmark:** "Cre8ve.xyz"
*   **Visual Features:** The "C" is designed as a smooth gradient mark, modern and sleek. The rest of the wordmark is neutral/white.
*   **Variations:** Icon version (just the 'C' or '8') and favicon versions tailored for both dark and light backgrounds.

## 4. UI Design Language
*   **Theme:** Dark AI interface design system.
*   **Core Elements:** Buttons, Cards, Floating UI panels.
*   **Visual Effects:** Glassmorphism (blurred backgrounds behind panels), Soft neon borders, and glow accents/UI highlights.

## 5. Website Design System
Cinematic landing page structure restricted to **ONLY four sections**:
1.  **Hero Section:** High impact, introducing the brand.
2.  **Capabilities Section:** Grid display of features.
3.  **AI Architecture Section:** Technical or structural showcase.
4.  **Call-to-Action Section:** Final conversion point.

## 6. Capability Card System
Structured cards for modeling capabilities. Standardized layout for:
*   AI Agents
*   AI Applications
*   AI Infrastructure
*   AI Automation
*   AI Interfaces
*   AI Integrations

## 7. Iconography
*   **Style:** Minimal futuristic thin line icons with subtle glow accents.
*   **Use Cases:** AI Agents, Automation, AI Infrastructure, Data pipelines, AI Interfaces, Workflow orchestration.

## 8. Motion Design
Animations should feel frictionless, smooth, and cinematic.
*   **Core Movements:** Smooth scroll motion, floating UI objects.
*   **Particle Effects:** Neural particle movement.
*   **Interactions:** Hover glow interactions and subtle micro-interactions across components.

## 9. Visual Identity & Image Style Guide
*   **Core Motifs:** AI-inspired visual motifs.
*   **Imagery Type:** AI-generated images, AI invocations, AI sculptures, and glowing highlights.
*   **Themes:** Neural networks, abstract glowing 3D shapes (spheres, brains), and futuristic androids/robotics with glowing accents.
